# Stackmob Custom Code SDK (Release Notes)

* Added additional query conditions
  * <, <=, >, >=
  * Geo queries
  * Null
  * Not Equal
* Order-by
* Pagination
* Expand queries
* Relational queries
* Get count of objects for a schema
* Atomic, conditional update one
